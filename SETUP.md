# DnD Adventure: The Whispering Ruins - Інструкція для запуску

## Про гру

Це інтерактивна текстова DnD гра з атмосферним Dark Medieval Grimoire дизайном. Гра включає:

- **Вибір персонажа**: Воїн, Маг, Слідопит з унікальними характеристиками
- **Розширений сюжет**: 20+ вузлів, які змінюються залежно від ваших виборів
- **Система бою**: Битви з ворогами з використанням кубиків d20
- **Інвентар**: Збір предметів, які впливають на сюжет
- **Система моралі**: Ваші вибори впливають на кінцівку
- **3 кінцівки**: Хороша, погана та нейтральна
- **Збереження гри**: Зберігайте та завантажуйте прогрес

## Вимоги

- Node.js 18+ (рекомендується 22+)
- pnpm або npm

## Встановлення

1. **Клонуйте або завантажте проект**
   ```bash
   git clone <посилання-на-репозиторій>
   cd dnd-adventure-game
   ```

2. **Встановіть залежності**
   ```bash
   pnpm install
   # або
   npm install
   ```

## Запуск

### Режим розробки
```bash
pnpm dev
# або
npm run dev
```

Сервер запуститься на `http://localhost:3000`

### Виробничий режим
```bash
pnpm build
pnpm start
```

## Гра

### Як грати

1. Виберіть клас персонажа (Воїн, Маг, Слідопит)
2. Слідуйте сюжету та робіть вибори
3. Кидайте кубик d20 для перевірок атрибутів
4. Битися з ворогами або переговорюйте з ними
5. Збирайте предмети та золото
4. Досягніть однієї з трьох кінцівок

### Система моралі

Ваші вибори впливають на систему моралі:
- **Позитивні вибори** (+мораль) → Хороша кінцівка
- **Негативні вибори** (-мораль) → Погана кінцівка
- **Нейтральні вибори** (0 мораль) → Нейтральна кінцівка

### Збереження гри

Натисніть кнопку **"💾 Меню"** у верхньому правому куті, щоб:
- Зберегти гру
- Завантажити збережену гру
- Видалити збереження
